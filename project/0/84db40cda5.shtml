“柯洁”们带火的棋类培训市场还能火几年？

“柯洁”们带火的棋类培训市场还能火几年？
2020年06月08日 12:44 澎湃新闻
缩小字体 放大字体 收藏 微博 微信 分享 0
腾讯QQ QQ空间

　　导语

　　2016年到2017年，阿尔法围棋（AlphaGo）先后和包括围棋世界冠军柯洁、李世石在内的国内外数十位围棋高手对决，几乎无一败绩。人类虽不如AlphaGo战绩瞩目，但国内依旧掀起了一阵围棋热潮。一时间，无论圈内圈外围棋的话题热度居高不下。后AlphaGo时代，围棋等棋类培训项目层出不穷，近几年成为了资本的关注重点。

　　随着互联网时代发展，部分棋类培训、赛事可以通过网络进行。在今年疫情影响下，棋类运动成为为数不多的、可以全民参与的活动。

　　01

　　棋类运动项目

　　4000多年前，围棋和五子棋出现，但具体哪一个发明的更早，目前已无从考证。围棋，中国古代称为“弈”，关于围棋的起源，最流行的说法是“尧造围棋说”。但除此之外，还有“战国造围棋说”、“乌曹造围棋说”和“异域传入说”等说法。

　　人类历史上有过无数种棋类游戏，能经久不衰且在世界范围内广泛普及的有国际象棋、国际跳棋、五子棋、中国象棋、围棋等。这些都属于国家体育总局认可的智力体育运动。五棋当中，国际象棋、中国象棋、围棋最为流行，职业化程度最好，被称为三大棋。

　　02

　　发展历程

　　三大棋盛行

　　据《左传》中记载，大叔文子曰：“今宁子视君不如弈棋，其何以免乎？”，表明围棋在春秋时期已经广为流传。此时还流行一种叫做六博的古代棋戏。“博弈”一词中的“博”就是指六博棋，“弈”是指围棋。战国时，屈原的作品《楚辞·招魂》中有“蓖蔽象棋，有六薄些”的词句。用象棋一词指六博始见于此。

　　东汉初年“博行于世而弈独绝”，直至东汉中晚期，围棋活动才又渐盛行。唐朝时期设立“棋待诏”一职，用以招揽国内（甚至包括国外）的围棋高手。棋待诏的设立，大大提高了棋手的社会地位。在科举考试之外，士人和平民又多此一条进身之阶，有许多人倾毕生之精力研究棋艺，成为职业棋手。

　　后汉明帝时，张骞出使西域，中国与印度开始正式交往，打通了陆路交通，同时开启了棋类文化的往来。到隋唐时期，中国与印度等地交往更加频繁。目前得到世界公认：国际象棋起源于亚洲，后来传入欧洲。11世纪末叶，遍及欧洲各国。在当时的文献中，将国际象棋列为骑士教育的 “七艺” 之一。“七艺”是骑术、游泳、射箭、击剑、狩猎、赋诗和下棋。

　　棋类运动定型

　　到了宋代，宫廷设的“ 棋待诏”中，象棋手占一半以上，民间还流行“大象戏”。北宋时期，先后有司马光的《七国象戏》，晁补之的《广象戏图》等著术问世。经过近百年的实践，象棋于北宋末定型成近代模式。南宋时期，象棋家喻户晓，成为流行极为广泛的棋艺活动。李清照、刘克庄等文学家，洪遵、文天祥等政治家，都嗜好下象棋。

　　明、清两朝的翰林院虽依旧置有“待诏”，但已不见“棋待诏”这种名目。围棋解除了士大夫的垄断，在市民阶层传开，通过频繁的民间比赛活动，使得围棋游艺更进一步得到了普及。围棋在清代得到了高度发展，名手辈出，棋苑空前繁盛。

　　同一时期，国际象棋和国际跳棋在国外定型。国际跳棋由各国的民族跳棋演变而来。据史学家研究，跳棋起源于古埃及、古罗马、古希腊等一些国家和地区。但是由于学术的不同看法，考古学家们还无法准确地推断出古跳棋的起源地和传播途径。

　　1688年，中国古代的五子棋经由高丽（朝鲜）传到日本。起先，五子棋是宫廷贵族的游戏，后来流传到了民间，迅速发展，为人们所深爱。据说爱好者人数一度超过了围棋。到了公元1899年，五子棋命名为“连珠”。从此，五子棋运动走上了一条不断改良的道路，通过对一系列规则的改良，使五子棋这一简单的游戏复杂化、规范化。

　　近代之后：棋类运动正规化、培训产业兴起

　　近代之后，随着列强的入侵，棋类运动也随着中国社会低潮一起开始没落，少有精力开展和组织活动。1952年，北京棋艺研究会成立，这是中国第一个专业的围棋机构，也是我国围棋走上正规化、组织化、系统化道路的开端。

　　1956年，围棋、象棋、国际象棋正式被定为国家开展的体育项目。1962年，中国围棋协会在安徽合肥市正式成立，总部设在北京。同年，中国象棋协会成立，各地相应建立了下属协会机构。1978年，中国首次参加国际象棋奥林匹克世界团体锦标赛，在世界范围内崭露头角。

　　1984年，第一届中日围棋擂台赛开幕。中国人聂卫平在前三届擂台赛中获取11连胜，极大的推动了中国围棋的普及，掀起了一场空前的围棋热。各地棋社应运而生，有官办的有私办的。让孩子学下棋，进专业队，成了多少父母的心愿，一时间棋手成了令人羡慕的职业。1991年，中国棋院成立，围棋联赛启动，各类围棋赛事如火如茶开展，围棋学校、围棋教室、围棋会所也如雨后春笋散播开来，预示着产业化的发展时代的到来。

　　2001年2月23日，国家教育部体育卫生与艺术教育司和国家体育总局群体司联合下发“关于在学校开展围棋、国际象棋、象棋三项棋类活动的通知”。通知指出：“各级教育、体育行政部门在促进学生全面发展的素质教育过程中，要结合本地区和各学校实际，作为‘体育与健康‘课程中课外文体活动。”2003年在苏州举办的百校国际象棋进课堂国际研讨会上，命名了首批82所已经获得教育部体卫司及国家体育总局棋牌运动管理中心联合授牌的全国国际象棋进入校园的传统学校。国际象棋在校园中得到了较好的普及和推广。

　　2006年2月，五子棋成为我国正式开展的体育项目。2008年，第一届世界智力运动会在北京举行，国际跳棋、国际象棋、围棋、象棋作为参赛项目博得了大众关注。2009年，中国国际跳棋协会成立。2014年，“北京中小学棋牌运动教育推广工程”项目启动。在北京市中小学广泛普及棋牌运动。

　　2015年，全国棋牌工作会议上，国家体育总局棋牌中心党委书记杨俊安指出，接下来我国棋牌工作的重点是构建“培训、认证、注册、竞赛”四个棋牌公共服务体系，打造一个综合性网络服务平台。校外相应培训机构增多，围棋、国际象棋等依旧是培训热点。

　　03

　　棋类培训赛道现状

　　1。棋类培训赛道融资情况

　　对于棋类运动而言，企业投资最多的领域，多在各大型比赛、竞赛，如三星杯世界围棋公开赛、应氏杯世界职业围棋锦标赛等等。

　　据互联网披露的数据显示，2007年至2020年3月，棋类培训赛道中共发生14起融资事件。过千万融资事件两起，分别是聂卫平围棋道场于2017年9月获得的天使轮2000万人民币融资和真朴围棋于2017年3月获得的A轮千万级美元融资。半数融资事件未披露融资金额。

　　2。棋类培训项目融资轮次分布

　　从轮次分布上来看，天使轮融资事件最多，为7起，股权融资其次，为3起。融资主要集中在Pre-A轮以前。行业出现较早，但赛道内融资依旧处于刚刚起步的阶段。

　　3。棋类培训项目融资时间、种类

　　从融资时间上，我们可以看出，除2007年发生的一起天使轮融资外，其余融资事件都在2016年后发生，主要因为AlphaGo效应突出。到目前为止，有关于棋类的培训班种类有很多，但是真正可以吸引到融资的项目主要还是围棋。

　　4。棋类培训项目融资地域分布

　　从地域分布上来看，投资地域分布以北京为主，共有六起项目。此外浙江两起，西安、湖南各一起。

　　5。关注棋类培训项目的投资机构

　　我们对出手投资棋类培训项目的机构进行了统计。其中，北塔资本和真格基金均先后两次投资该领域。好未来亦有过出手参投。

　　04

　　棋手考级、教练培训逐步规范

　　在棋类领域中，不同棋类的级、段称号各有不同。1981年国家体委颁布了《围棋棋手段试行标准》，并颁发了首批高段证书，围棋运动开始成为一项正规化的运动。

　　在围棋定段赛中，参赛选手年龄要求在25岁以下。在全国国际象棋少年冠军赛中，8岁组最高申请称号为棋协一级棋士，10岁组最高申请称号为棋协候补大师，12岁及以上组最高申请称号为棋协大师。象棋和国际象棋等级称号部分区分男女。大多数靠比赛名次或胜率获取相应等级。以国际象棋为例，据中国国际象棋协会统计数据显示，2010年全国大型国际象棋赛事有54起，2018年有114起，赛事数量翻倍。棋类运动流行趋势明显。

　　2010年开始，各协会组织开办教练培训班，为期2-3天，加强教练员业务水平，提高各棋类培训机构教学质量。2019年7月，教育部等六部门发布《关于规范校外线上培训的实施意见》，要求校外线上培训机构具有完善的招聘、审查、管理培训人员的办法，师资队伍相对稳定。对于学科类知识培训的人员，应当具有国家规定的相应教师资格。目前棋类培训教练并没有明确要求需要取得教师资格证。

　　05

　　磕磕绊绊中发展

　　棋类运动中的刻板印象

　　在社区凉亭或公园里，经常可以看到一群大爷在下象棋、聊天。导致许多家长对于象棋存在着刻板印象，觉得太过于接地气。此前，在短视频平台上，一批年轻人各自发起了一项“挑战象棋大爷”的活动，打破了象棋在大众眼中的年龄壁垒，起到了很好的推广宣传作用。同样的刻板印象也存在于五子棋中。五子棋门槛较低、上手简单，是许多人茶余饭后的小游戏，很难将其与竞技项目挂钩。竞技五子棋中的规则不被大众熟知，相关培训也就更难开展。国际象棋和国际跳棋比象棋、五子棋、围棋更加“新颖”，在我国发展时间虽然不及其他棋类，但是发展速度迅猛，尤其是国际象棋，已经在我国棋类培训中占有一席之地。

　　2001年教育部下发了《关于在学校开展“围棋、国际象棋、象棋”三项棋类活动的通知》。2014年，“北京中小学棋牌运动教育推广工程”项目启动。2015年，全国范围内开始开展中小学棋牌运动教育推广工程。在中小学开展围棋、象棋、国际象棋、桥牌、国际跳棋、五子棋五棋一牌的教育推广，以新颖有效的推广方法、简捷多元的教育形式，让广大中小学生掌握棋牌运动技能。

　　校内五棋推广同步进行，但校外相关培训班的热度却不尽相同。棋类运动与智力培训挂钩，不少节目中出现的学霸对围棋颇为精通、围棋凭借专业棋手与AlphaGo对战，一度成为棋类培训中家长的首选。

　　小份额的成人棋类培训市场

　　目前，无论是哪类棋种，少儿培训都占据了大部分份额，其多以兴趣为主导、以启蒙或考级为目的。成人棋类培训多为基础、兴趣班。专业棋手几乎都是“年少成名”，部分竞赛选手年龄要求25岁以下，这也在一定程度上限制了成人棋类培训的发展。

　　由于目前少儿接触到棋类运动的时间越来越早，参与定段赛的棋手年龄越来越小，部分成人棋手在刚开始参加定段赛时，在一群孩子中间总显得格格不入。赢了有时会被孩子家长说胜之不武，输了面子上又过不去，年龄的差距劝退不少成年人，甚至有人会选择直接买一个证。

　　但同时，成人棋类爱好者潜力非常大。一些小学员在上课时，家长会和孩子一同上课。部分培训机构也以此为契机，推出亲子类课程项目，或拓展真正以成人为主体的培训业务。

　　没有分数加成后，为应试教育让路

　　2014年，教育部发布公告，就进一步减少和规范高考加分项目做出规定，公告中取消了体育特长生加分项目。打消了部分考生以棋类运动作为高考加分项的念头。此举是为将特长生学习的东西，变为人人都学的项目，防止体育美育功利化的同时，促进相关素质教育发展。

　　然而，随着学生年龄增长，课业压力越来越繁重，棋类运动没有了加分“价值”之后，也就不得不为应试教育让路。可支配给棋类运动的时间微乎其微。以围棋为例，世界冠军年龄越来越低，取得成绩后的小运动员们未来如果想要专心搞事业、更有进益，就免不了放弃学校里的学习时间。二者选其一，真正取得全国乃至世界顶尖成绩的少之又少，选择结果一目了然。

　　名师噱头，真假难辨

　　以2016年闹的最沸沸扬扬的“聂卫平围棋教室”为例，不少培训机构假借“聂卫平”之名宣传。2016年10月聂卫平本人亲自出面，发布声明称，“聂卫平围棋教室”商标已不再使用，与“聂卫平围棋道场”无关联。随后相关机构宣布更名，但有不少家长发现，其宣传招生时依旧打着“聂卫平”的旗号，并且截至目前，部分机构依旧沿用“聂卫平围棋教师”，并无改正。

　　以职业棋手的名义开办的培训班在一定程度上保障了师资力量，但同时也存在只挂名、偶尔客座或被盗用的情况。消费者在选择时，一时无法辨别真伪，就容易落入商家的圈套。

　　在线教育风口已到

　　随着“互联网+”时代的到来，网络与我们的关系越来越密切，棋类培训也逐渐拓展到线上，以直播课+录播课形式进行。部分机构还会附赠棋盘棋子，供学员在线下自主练习。线上发展给棋类培训带来了新机遇。高水平的职业棋手和名师大多集中在大城市，中小城市或低线城市的学员很难亲赴大城市学习棋艺。然而，中小城市棋类市场庞大，部分城市棋类教育水平有限，学员对棋类培训有需求。棋类项目形态，十分适合在线教育。虽然现在家长对于在线学棋的方式还不太熟悉，但是肯定会逐渐适应和接受。在线教育具有交通便利和师资优势，用户体验也会越来越好。在线教育规模扩大之后能否保证教学质量，师资和管理能力将是未来最大的考验。

　　某在线机构围棋课程

　　受2020年疫情影响，竞技体育赛事几乎“全军覆没”，棋类比赛却正在如火如荼的进行当中。4月，刚刚结束的LG杯中国预选赛，是围棋个人世界比赛首次通过网络进行，中国围棋队也成为“第一个吃螃蟹的人”。“加油中国”2020年中国体育彩票全国象棋业余棋王赛网络赛，截至4月初，线上参赛人数已经超过40万。各地区也有开展相应象棋、围棋等网络竞赛，开创竞赛新模式。

　　结语

　　各大顶尖棋手和AlphaGo的对决，再一次带火围棋的同时，同样也引起了人们对编程、尤其是少儿编程的重视。两种课程的核心，都聚焦在对孩子逻辑思维的训练。2018年开始，已经有机构将棋类运动和少儿编程相结合，开发创新型课程。通过编程角度，理解各类棋种自动提子的判断算法，能够实现自动提子棋盘。但也有机构只是同时教授两门课程，并没有做到有机结合。少儿编程向棋类培训扩展相对容易，棋类培训向少儿编程靠拢边际成本较高，道路并不好走。有不少人会担心，未来的某一天少儿编程会抢了围棋甚至棋类运动培训的饭碗。但究竟如何，还未可知。
