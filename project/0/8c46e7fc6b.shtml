昆陆领队:水中踢球男孩其实很有天赋 只是接触太晚

昆陆领队:水中踢球男孩其实很有天赋 只是接触太晚
2020年06月08日 10:16 国内足球综合
缩小字体 放大字体 收藏 微博 微信 分享 0
腾讯QQ QQ空间

　　稿件来源：足球报

　　记者贾岩峰报道 在没有联赛的2020年，足球圈能够引人关注的新闻十分寡淡，就在最近，一个来自云南西双版纳的布朗族小伙子火了。他在水中练习足球的视频迅速传遍网络，而云南昆陆队邀请他试训后更是让他成为了整个足球界关注的一个热点。他会不会成为一个从业余球员身份直接登陆职业联赛的传奇？然而这几天网络上不断爆出“岩坎香试训失败，只能返回学校继续读书”等字眼，那么岩坎香真的失败了吗？职业队对他的评价又是怎样的？上周五上午，记者联系到了云南昆陆的和云群领队，并且在他的协助下也采访了岩坎香本人，听他们讲述了小岩这些天试训的过程以及小岩的足球梦。

　　15岁才开始触球的少年

　　岩坎香出生于2000年，马上就年满20岁了，来自云南省西双版纳州勐海县布朗山布朗族乡曼囡村的他，甚至普通话都还不算特别流利，但他却非常认真地回答着记者的每一个问题，如果没有听懂就会要求记者再慢点重复一次，记者也尽量用比较简单的问题去提问。但是记者问完第一个问题，就被他的回答惊到了。“你首次接触足球是什么时候？”“15岁。”记者以为自己听错了，重复一次再问他，是四五岁，还是十五岁。他说：“刚上初中那年，15岁的时候，开始知道还有足球这项运动存在，我从那时候才开始练习足球。”也就是说，从他接触足球，到现在，也就刚刚五年的时间，但是看他的身材和气场，像是练习了很多年的样子。

　　小岩今年在上高二，明年才是他的高考年，在接到云南昆陆俱乐部的试训电话之前，他一直都在努力地为自己的高考计划准备，尽管他也渴望成为职业球员，但他也觉得能够考上一所有足球专业的二本大学可能更现实。“为什么是二本？不能是一本？”记者对于他这个目标很好奇。“因为布朗族这些年能够考上大学的都是三本居多，二本很少很少，反正我没怎么听说过。”岩坎香很坦率地表示，如果自己能够考上二本，就足够让父母和他自己开心了，而村里的人也会为他感到骄傲。

　　考与足球相关的专业的二本大学，除了要拿到相应的文化课分数外，还要有足球个人专项技术考核，而岩坎香之前很多训练也都是围绕着这些考试内容进行的，而且对于拿到高分也已经很有把握了。“有那么几项，带球绕桩，射门入网，还有把球射进指定的圆心内，前两项我都没问题，第三项五个球我差不多能有两到三个直接踢进圆心。还有时间，我可能还能练得更好一点。”小岩听起来很有信心。所以还有一年多的时间可以让他去强化训练，而如今接到职业队的试训邀请电话，如果能够留下，是不是他的高考计划就要改变了呢？

　　“我可能还是要回去准备高考，但我还是会在这里完成一周的训练，俱乐部对我很好，我非常感谢。”小岩告诉记者，因为在过去这些年里，他也没有教练，也没有系统的足球战术训练，体能就更谈不上，来到职业队后，第一天就仅仅是一个训练前的热身运动跟下来后都觉得呼吸困难，他真正体验到了职业与业余之间的差距，而云南昆陆的和领队说，小岩其实非常有天赋，而且有韧劲和执着精神，只是他接触足球时间太晚，而且训练不系统，所以没有办法马上跟上职业队的节奏，但是小岩的到来对于昆陆队来说是一种激励，因为小岩身上的质朴与对足球那种既单纯又猛烈的热爱，对现在能够踢上职业联赛的球员们来说，让他们体会到了自己拥有的实际上是许多像小岩这样的少年渴求却难以得到的。

　　“我们不同意网上的说法，说什么小岩试训失败，水平不行。这对小岩不公平，也根本不了解昆陆俱乐部的初衷。”云南昆陆俱乐部的和领队告诉记者，当初邀请小岩来，是不想埋没他的足球梦，如果能够留下他是最好的，但是小岩尽管目前还不适合直接加入职业联赛，但小岩带给昆陆的正能量以及他对足球的执着，还有他独特的训练方式对他身体素质的提升，这些其实也都深深震撼了职业足球人，小岩没有失败，因为足球本身除了成败还有就是信念的传递，永不言败精神的传递，小岩虽然不做职业球员，但是他还会一直在职业足球的道路上走下去，以后也许会以另外一种身份和方式进入职业足球领域，这个谁都不能断言肯定不会发生，因此不能说他就是失败了。

　　岩坎香练球的故事

　　对于我们城市长大的孩子们来说，即便父母再进行网络和电视封锁，也不太可能到初中才知道有足球这项运动的存在，那么岩坎香为什么会有这样的经历？其实这个多少与他所处的地区和环境有一定关系。岩坎香所在的布朗族，其实属于云南省的九大“直过民族”之一， “直过民族”特指新中国成立后，未经民主改革，直接由原始社会跨越几种社会形态过渡到社会主义社会的民族，他们所居区域被划定为“民族直过区”，对他们采取了不同于其他地区的特殊政策，使他们跨越若干社会发展阶段，得以与其他民族一起迈步在社会主义道路上，实现各民族共同繁荣进步。

　　“我们村的人都很质朴，我父母都是农民，他们也不知道足球是什么，只是知道我喜欢，支持我。“

　　岩坎香小学设在村委会里，那里条件有限，但水泥地上的篮球架带给了他无限的欢乐，小学时代打篮球是他最大的爱好，而那时候他根本不知道有足球这项运动。“等我上了中学后，到了镇里读书，知道有足球了，我马上就放弃了篮球，疯狂地喜欢上了足球。”岩坎香说，他所在的民族中学有很多学生，但真正踢球的却不多，为什么呢？“因为足球场在水泥地上，踢球太容易受伤了，虽然篮球场也在水泥地上，但是受伤的可能性肯定比足球小。”因此，在初中时代，篮球依然是学校第一运动，足球排不上。那么小岩不怕受伤吗？“我不怕，我从小就在村里摔打惯了，自己学会控制一下身体就好了。”岩坎香告诉记者，身高一米六八的他从小就很灵活，身体协调性很好，虽然在水泥地上踢球危险性不小，但越是有难度他就越是喜欢。

　　在自己体会到了足球的快乐之后，他就想办法把这种快乐带给同村的小伙伴们。“我觉得足球是一个能够让更多人参与，然后带给所有人快乐的运动，我跟学校借了球，带回到村子里跟大家一起玩儿。”岩坎香所在的村子经济条件有限，很多小伙伴都不知道足球是什么，所以当岩坎香第一次教大家踢球的时候，小伙伴们也都激动得够呛。“我们那里是山区，很少有平地，即便有，也都是石头或者杂草，想找到适合踢球的地方太难了，但即便是那样，小伙伴们也都能克服，这就是足球的魔力。”岩坎香说，他们那时候也没什么像样的服装和鞋子，衣服就是随便穿，很多时候都光脚踢，被扎到或者踢出血那是常事，但是后来脚底板都磨出厚厚的茧子来就什么都不怕了。没有教练，他就是大家的教练，而他也喜欢那种教大家踢球的感觉。

　　自从岩坎香接触足球后，带着他训练和踢球的，只是中学的体育老师，老师本身也不是从事足球专业的，因此能够传授给他的足球知识与技能也比较有限，但是这挡不住岩坎香渴望学习的心，上了高中后他有了手机，于是利用手机搜索一些训练方法，现学现卖他就传授给了小伙伴们。“我们不太懂什么是整体战术，只是知道尽量把球控制在自己身体范围内，然后传球，盘带的动作也都是跟着一些视频剪辑去模仿的，还有任意球，我们用轮胎练，看谁能够穿过轮胎中心把球踢出去。”在岩坎香发布的视频中，他带着小伙伴们从摩托车的上方跳过去后然后踢球，或者夹着球从车身上跃过去，能够看到他有很好的弹跳力和爆发力。球门非常简陋，是他用平时干农活的砍刀砍了一些枯枝搭成的，可正因为如此，大家才格外开心。

　　在岩坎香穿的一些运动服中，15这个数字非常醒目。这个15对他而言，就是他认识足球的年龄，他觉得这个是值得永远纪念的。在学校上课的时候，有时候会把一些废报纸用透明胶粘起来，做成足球的形状，然后他在走廊里拿来练习，因为这样落地或者飞出去也不会打扰到同学，更不会砸到谁，所有除了必须坐在教室里读书以外的时间，他都想尽可能地留给足球。“因为我回到家里之后，还要干一些农活，收割、种田、给茶树喷农药，把这些都做完了之后我才能去踢球。”岩坎香还有一个妹妹，跟他在同一所学校同一个年级，当记者说他是家里主要劳动力时，他说事实上妹妹干起农活来比他还厉害，有时候为了多给他挤出一些练球的时间，甚至要承担更多的家务，令他很感动。

　　那么岩坎香为何要在水里练球呢？是为了吸引更多眼球，成为网络红人？还是有别的原因？“我从来没想过当什么网络红人，在水里踢球原因很多，第一，凉快，第二可以练习爆发力，第三，能够做一些危险的动作，比如说凌空射门，在我们那里的土地上就很容易受伤，但是摔到水里就好很多。”但是小岩他们的确把视频传到网上只是想要留个纪念，因为手机万一坏了视频就可能都没了，但是如果都传到网上，就能够永远保存下来。“我没想到自己会收到昆陆队的邀请，我甚至无法相信自己的耳朵听到的是事实，反复确认后我才相信，我真的有机会去亲眼看一下职业队了。”小岩在出发前特意花光了自己的全部积蓄——300元买了一双足球鞋，带着小伙伴们的祝福和期望出发了。尽管最终没有能够留在职业队，但是这一段经历，也足够他跟小伙伴们说上三天三夜了。

　　岩坎香平时很少有机会能够看到国内外的职业联赛转播，也没有看过完整的国家队比赛录像，但是他也有自己喜欢的球星。“我最喜欢武磊，喜欢他的技术特点，不是凭借身高，但却凭借敏锐的嗅觉和天赋，以及后天的努力，尤其是他前插的那一下意识，让我由衷佩服。”小岩希望未来有一天能够现场看一次武磊的比赛，要是能够跟他交流一下就更完美了。

　　韦世豪给他寄来球衣球鞋

　　作为云南职业足球的独苗，云南昆陆足球俱乐部一直在低调中前行着。而这次邀请岩坎香试训，让更多人把关注的目光投向了这家中乙俱乐部。说起邀请岩坎香试训的初衷，和领队告诉记者，一切都是源自于网上的视频以及看到了小岩的心愿，所以就想给他一个追梦的机会。“我们是是在当地足协的帮助下联系上小岩的，然后跟他约定好了一周的试训时间，这一周时间很快，我们都很喜欢这个小伙子，他带给了我们不太一样的足球世界，也让我们坚信，昆陆一直坚持的足球理念是对的。不仅要追求成绩，更需要关注足球本身的社会和文化教育功能，追求足球本身的快乐与积极，小岩就让我们看到了这些希望。”尽管岩坎香在个人综合实力上，距离职业球员的平台还差得较远，但他的这种执着精神，其实正是提升中国足球所需要的。

　　岩坎香没有受过专业的训练，从基本功以及整体战术素养上，跟职业队都有很大的差距，他自己也承认，职业足球远比看到的转播中更难踢。“平时大家批评职业足球挺多的，都觉得职业球员踢得臭，这次我也跟小岩沟通了，他也承认，其实很多足球的东西，看似简单，但真的要是比起来，普通足球爱好者们别说职业球员一半的水平，就是几分之一的水平也达不到。这次我们带给他的震撼是让他看到了职业足球这个领域其实有太多可以学和做的事情，这是他此行来最大的收获。而我们在小岩身上看不到他对游戏的沉迷，看不到他眼神中的迟疑和犹豫，能看到的只有对足球完全的投入。”岩坎香说，他无比渴望考上昆明的某所大学，不离开云南，能够继续追随昆陆俱乐部。

　　在和领队看来，中国足球非常需要有更多的“岩坎香”，需要能够放下手机，忘记电子游戏，去全身心体验足球快乐的青少年，同时也愿意用自己的力量，去感染身边更多的人加入到足球的快乐中来，无论生活环境如何，面对多少困难，都愿意用一种积极向上的态度去面对。不是每一个球员都一定能够走职业化道路，但是真正强大的职业化足球，恰恰需要更多来自草根和业余足球的热爱，才能更好的升华。

　　据和领队介绍，岩坎香的故事感动了中超球员韦世豪，韦世豪特意给岩坎香寄来了球衣和球鞋，鼓励他不要放弃足球的梦想，哪怕不做职业球员，也要一直在热爱足球的路上一直走下去。

云南昆陆水中踢球岩坎香
